Sobre Mim

Com um histórico de bolsista de iniciação científica do PET Computação, minha jornada acadêmica na UFSC tem sido marcada por uma dedicação intensa à pesquisa e desenvolvimento. Meu conhecimento técnico abrange linguagens de programação como Python, Go, C e C++, habilidades que apliquei em diversos projetos. Atualmente, estou expandindo meu conhecimento em Computação Distribuída e aprimorando minhas competências práticas com a ferramenta Apache Kafka. Estou sempre em busca de novos desafios e oportunidades para contribuir e crescer na área de tecnologia.

Projetos

Gerador de Certificados

Programa em Python para gerar e enviar certificados de participação para eventos, como a SECCOM (Semana Acadêmica da Computação). O programa lê dados de arquivos CSV, gera um certificado no formato PNG e o envia por e-mail aos participantes.

Git Tutorials

Guia básico dos principais comandos do Git para iniciantes, o projeto contém um readme que elenca os principais comandos, para que cada comando serve e sua explicação.

Testes de Software

Projeto simples de um sistema de cálculo de salário, desenvolvido como parte de um estudo sobre Testes de Software. O objetivo principal deste projeto é demonstrar a aplicação de testes de nível de unidade utilizando a técnica de caixa preta, baseada em especificação.
